## 2. 卡尔曼滤波
1.卡尔曼滤波器如果不看K(卡尔曼增益），P（协方差矩阵），剩下的部分就是一个状态观测器，所以要理解卡尔曼滤波器的滤波作用可以尝试观察这个观测器的极点变化曲线。这些极点最后都会收敛到一个很小的范围内，所以卡尔曼滤波器可以简化成极点固定的状态观测器。
